安装todo示例(mongodb版)

  • 1、安装mongoskin模块，npm install mongoskin
  • 2、修改app/controller/todo-mongodb.js 里的 `服务器地址` 配置。
  • 3、访问：http://{{app.req.headers.host}}/todo-mongodb 查看示例。

安装todo示例(mysql版)

  • 1、把todo.sql导入你的mysql数据库。
  • 2、安装mysql模块，npm install mysql
  • 3、修改app/controller/todo.js 里的 `数据库帐号` 配置。
  • 4、访问：http://{{app.req.headers.host}}/todo 查看示例。
